# Plan 11-04: Map Overhaul — Borderless, Glowing Routes & Pulse Markers

## Wave 4 — Map (After Primitives)

<objective>
Overhaul the map component ecosystem to match the reference image's premium map presentation: borderless rounded-3xl container, glowing neon polyline routes with cyan-to-emerald gradients, pulse animations on start/end nodes and charging stations, custom dark map tiles, and glass-themed popups/overlays. The map should feel like a living, pulsing cockpit radar — not a static embedded widget.
</objective>

<context>
**Reference image — Map (Dashboard, top-left panel):**
1. **Container:** Rounded corners (`rounded-3xl`), NO visible border. The map bleeds seamlessly into the dark background. Only a subtle shadow separates it.
2. **Map tiles:** Ultra-dark theme — dark blue/gray landmass, barely-visible road network, muted labels. CartoDB dark-matter or Stadia dark variant.
3. **Route lines:** Thick glowing polylines with neon gradient. Main route: emerald-to-cyan gradient with glow underlay. Alt routes: dashed, lower opacity.
4. **Node markers:** 
   - Charging stations: Amber ⚡ icons with pulse animation (expanding ring)
   - Start/End: Green/Red dots with pulse glow rings
   - Network nodes: Very small, barely visible dots
5. **Map controls:** Small glass buttons (+/−) in top-left, semi-transparent
6. **Weather overlay:** Small glass toggle pill "☁ Weather" with on/off state
7. **Map popups:** Dark glass cards with route info — `bg-white/[0.03] backdrop-blur-2xl rounded-2xl`

**Reference image — Map (Route Planner, top-right panel):**
1. Charging stops marked with labels: "Charging Stop / Estim Charge Time"
2. Route drawn between stops with thick green/teal line
3. Multiple stops connected

**Reference image — Map (Analytics, bottom-right panel):**
1. Heatmap overlay with gradient: green → yellow → red (charging demand hotspots)
2. Semi-transparent heatmap blending with dark tiles

<task id="2">
  <name>Upgrade mapStyles.ts — single dark theme + route colors</name>
  <files>frontend/src/components/map/mapStyles.ts</files>
  <action>
    Simplify to dark-only styles. Remove light theme entirely.

    **New mapStyles.ts:**
    ```ts
    import type { StyleSpecification } from 'maplibre-gl';

    // CartoDB Dark Matter — ultra-dark tiles
    export const DARK_STYLE: StyleSpecification = {
      version: 8,
      sources: {
        'carto-dark': {
          type: 'raster',
          tiles: ['https://basemaps.cartocdn.com/dark_all/{z}/{x}/{y}@2x.png'],
          tileSize: 256,
          attribution: '© OpenStreetMap contributors, © CARTO',
        },
      },
      layers: [{ id: 'carto-dark-layer', type: 'raster', source: 'carto-dark' }],
    };

    // Offline fallback — solid midnight
    export const DARK_OFFLINE_STYLE: StyleSpecification = {
      version: 8,
      sources: {},
      layers: [{ id: 'bg', type: 'background', paint: { 'background-color': '#0a0f16' } }],
    };

    // Route colors — neon palette
    export const ROUTE_COLORS = {
      primary: '#10B981',    // emerald
      alt1: '#14B8A6',       // cyan/teal
      alt2: '#F59E0B',       // amber
      alt3: '#3B82F6',       // blue
      alt4: '#EF4444',       // rose
      glow: 'rgba(16, 185, 129, 0.25)',  // emerald glow underlay
    };

    // Always return dark
    export function getMapStyle(online = true) {
      return online ? DARK_STYLE : DARK_OFFLINE_STYLE;
    }
    ```

    **Removed:**
    - `LIGHT_STYLE` — no light mode
    - `LIGHT_OFFLINE_STYLE` — no light mode
    - Any theme-dependent logic
  </action>
