# Plan 11-01: Core CSS Tokens, Design System & Bento Grid Setup

## Wave 1 — Foundation (No Dependencies)

<objective>
Nuke the entire existing CSS variable system (1,211 lines of index.css, 294 lines of tailwind.config.ts, theme-cycling scripts in index.html) and replace with a minimal, flat design token system purpose-built for the ultra-premium bento UI. Establish the deep midnight slate background, ultra-translucent glass surface tokens, neon accent palette, radial gradient overlay, and CSS Grid bento utilities. After this wave, every subsequent plan builds on a single source of truth for colors, spacing, and card surfaces.
</objective>

<context>
**Reference image — Design DNA (all 4 panels):**

The reference shows a hyper-modern cockpit UI with these precise visual properties:

1. **Background:** Deep midnight slate `#0a0f16` — not pure black, not navy. A cold dark blue-gray.
2. **Radial overlay:** Subtle radial gradient from center (`rgba(16,185,129,0.03)` → transparent) adding depth to the flat bg.
3. **Card surfaces:** Ultra-translucent, barely visible — `bg-white/[0.02]` to `bg-white/[0.04]`. Cards are "floating dark voids" with faint frosted edges, NOT opaque panels.
4. **Card borders:** Ghostly thin — `border border-white/[0.05]`. Just enough to delineate edges.
5. **Card radii:** Large — `rounded-2xl` (16px) for standard cards, `rounded-3xl` (24px) for map containers.
6. **Backdrop blur:** Heavy — `backdrop-blur-2xl` (40px) for that ultra-premium frosted glass depth.
7. **Shadows:** Cards have a faint dark shadow: `shadow-[0_8px_32px_rgba(0,0,0,0.3)]`. On hover, an accent glow: `shadow-[0_0_20px_rgba(16,185,129,0.15)]`.
8. **Typography:** Inter font family. Labels in `text-slate-400` (muted). Values in `text-white` (stark). Massive contrast ratio between label and value.
9. **Accent palette:**
   - Primary: Emerald `#10B981` — nav active, buttons, values, progress fills, route lines
   - Emerald dim: `rgba(16,185,129,0.10)` — icon/badge/step backgrounds
   - Amber: `#F59E0B` — warnings, alternative metrics, SG-GAN chart line
   - Cyan: `#14B8A6` — chart lines, secondary data, teal accents
   - Red: `#EF4444` — errors, loss curves, stop buttons
   - Blue: `#3B82F6` — info badges, raw reward line
10. **Bento grid:** Cards laid out in a 12-column CSS grid with `gap-4` (16px) or `gap-6` (24px). Cards span variable columns/rows creating a mosaic "bento box" effect — NOT uniform grids.

**Current state (to be nuked):**
- `index.css`: 1,211 lines — `:root` vars, `.dark` overrides, `.glass-*` layers, `.btn-*` layers, `.input-*` layers, `.badge-*` layers, `.skeleton-*` layers, MapLibre overrides, presentation mode, custom keyframes
- `tailwind.config.ts`: 294 lines — `primary` (cyan), `accent` (amber), `surface` (blue-gray) 10-stop color scales, `.glass-bg/border/blur/surface` plugin utilities, 20+ keyframes, gradient backgrounds
- `index.html`: 33 lines — theme/presentation FOUC prevention script that can cycle `class="dark"`
- `App.tsx`: Uses `useTheme()`, `isDarkMode`, dark class toggling, `transition-colors duration-300`
- Zustand store: Persists `themeMode`, has `useTheme` selector

**What gets nuked:**
- ALL CSS custom properties (`--color-*`, `--glass-*`, `--focus-*`, `--text-*`, `--spacing-*`)
- ALL `.dark` overrides
- ALL `.glass-*`, `.btn-*`, `.input-*`, `.badge-*`, `.skeleton-*` component classes
- ALL `dark:` Tailwind prefixes across the codebase

- `useTheme()` hook calls and `themeMode` persistence
</context>
