2. **Create frontend/src/lib/geo.ts** — coordinate conversion and GeoJSON utilities:
   - `toLngLat(x, y, center?, spread?)` — converts abstract graph coords (0-100) to [lng, lat] (GeoJSON order). Same math as existing toLatLng in NetworkMap.tsx but returns [lng, lat] not [lat, lng]. Default center [30.2672, -97.7431] (Austin, TX), default spread 0.045.
   - `networkToGeoJSON(network: RoadNetworkData, center: [number, number])` — converts full RoadNetworkData to { nodesGeoJSON: GeoJSON.FeatureCollection, edgesGeoJSON: GeoJSON.FeatureCollection }. Nodes are Points with properties { nodeId: number, isCharging: boolean }. Edges are LineStrings from source to target.
   - `routeToGeoJSON(route: Route, posLookup: Record<string, [number, number]>)` — converts a Route to a GeoJSON Feature LineString. The posLookup values are already in [lng, lat] order. Returns { type: 'Feature', properties: {}, geometry: { type: 'LineString', coordinates } }.
   - `buildPosLookup(network: RoadNetworkData, center: [number, number])` — builds the Record of nodeId to [lng, lat] coordinates.
   - Import types from '../services/api'.
   - Export all functions.

3. **Create frontend/src/components/map/mapStyles.ts** — MapLibre style definitions:
   - Import StyleSpecification from maplibre-gl.
   - Export DARK_STYLE: StyleSpecification — CartoDB dark_all raster tiles with subdomains a/b/c, @2x, tileSize 256, source ID 'carto-tiles'. Attribution: '© OSM © CARTO'.
   - Export LIGHT_STYLE: StyleSpecification — CartoDB light_all raster tiles, same structure, same source ID 'carto-tiles'. **Same source ID is critical for styleDiffing to preserve dynamic layers.**
   - Export ROUTE_COLORS = ['#10b981', '#6366f1', '#f59e0b', '#ef4444', '#8b5cf6'] (same colors as current NetworkMap.tsx).

4. **Create frontend/src/hooks/useMapTheme.ts** — hook that returns current map style:
   - Import useSystemStore from store (access isDarkMode).
   - Import DARK_STYLE, LIGHT_STYLE from '../components/map/mapStyles'.
   - Return { mapStyle, isDarkMode } — mapStyle is the StyleSpecification based on isDarkMode.
   - Memoize the return value to prevent unnecessary re-renders.
  </action>
