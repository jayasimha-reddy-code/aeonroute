  `python
  if config.demo_mode:

  Also add system_config["demo_mode"] = config.demo_mode for non-demo case (defaults to False).

  In _run_pipeline(), if demo_mode, call sys.load_models() BEFORE training steps to load existing checkpoint weights. This way the 5 epochs of training are fine-tuning, not training from scratch.

  Specifically, insert after step1_create_road_network():
  `python
  if sys.config.get("demo_mode"):
      self._update(15, "Loading model checkpoints (demo mode)")
      try:
          sys.load_models()
      except Exception:
          pass  # If no checkpoints exist, train from scratch
  `

  The rest of the pipeline stays the same — the GAN train, RL train etc. will run with reduced epochs.
