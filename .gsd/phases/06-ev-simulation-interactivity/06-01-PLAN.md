Create a requestAnimationFrame-based simulation hook that drives an EV along a route.

**Interface:**
`	ypescript
interface UseEVSimulationOptions {
  route: Route | null;
  posLookup: Record<string, [number, number]> | null;
  startSOC: number;           // percentage (0-100)
  batteryCapacityKWh: number; // e.g. 60
  speedMultiplier?: number;   // default 1, controls simulation speed
}

interface SimulationState {
  isSimulating: boolean;
  isPaused: boolean;
  isCharging: boolean;
  progress: number;           // 0.0 to 1.0
  currentPosition: [number, number] | null; // [lng, lat]
  currentBearing: number;     // degrees for marker rotation
  currentSOC: number;         // percentage
  currentSegmentIndex: number;
  currentNodeId: number;      // node ID at or just passed
  chargingProgress: number;   // 0-1 during charging
}

// Return: state + controls
{ state: SimulationState, start: () => void, pause: () => void, resume: () => void, reset: () => void }
`

**Algorithm:**
1. Convert route.path to coordinates array via posLookup
2. Pre-compute cumulative segment distances (haversine or euclidean from coords)
3. On start(): begin requestAnimationFrame loop
4. Each frame:
   - If isCharging: increment chargingProgress over 2.5 seconds, SOC fills to min(95, currentSOC + 30). When done, resume movement.
   - Compute elapsed time (minus paused time), map to totalProgress using speedMultiplier (~3 seconds per segment)
   - Map totalProgress to segment index + fractional position within segment
   - Linear interpolation between segment endpoints for currentPosition
   - Compute bearing: Math.atan2(dy, dx) between current and next coord, convert to degrees
   - Compute SOC: startSOC - (route.energy_kwh * progress / batteryCapacityKWh * 100)
   - When arriving at a node in route.charging_stops: set isCharging=true, pause progression
5. At progress >= 1.0: set isSimulating=false (complete)

**Key details:**
- Use useRef for rafId, startTime, pausedTime to avoid stale closures
- Cleanup: cancelAnimationFrame on unmount or reset
- Bearing calculation: Math.atan2(nextLng - curLng, nextLat - curLat) * (180 / Math.PI)
- Make charging stops detection use Set for O(1) lookup
- Clamp SOC to minimum 0
