       ```python
       from backend.app.config import settings

       @router.get("/system/config")
       async def get_system_config():
           return {
               "ok": True,
               "data": {
                   "osmnx_radius_meters": settings.OSMNX_RADIUS_METERS,
                   "osmnx_radius_km": round(settings.OSMNX_RADIUS_METERS / 1000, 1),
                   "osmnx_center_lat": settings.OSMNX_CENTER_LAT,
                   "osmnx_center_lon": settings.OSMNX_CENTER_LON,

               }
           }
       ```

    2. **Add `api.getSystemConfig()` to `frontend/src/services/api.ts`**:
       ```typescript
       interface SystemConfig {
         osmnx_radius_meters: number;
         osmnx_radius_km: number;
         osmnx_center_lat: number;
         osmnx_center_lon: number;
         max_training_episodes: number;
       }

       async getSystemConfig(): Promise<SystemConfig> {
         const res = await fetch(`${BASE_URL}/api/system/config`);
         const data = await res.json();
         return data.data;
       }
       ```
