<objective>
Upgrade every surface in the UI to premium physical material quality. Cards must feel like actual smoked glass panels floating above a deep surface. Borders must simulate directional light catching on beveled edges. Shadows must be heavy and cinematic.

**Current state:**
- Card uses `bg-white/[0.04] backdrop-blur-3xl border border-white/10 border-t-white/20` — decent but not ultra-premium
- Shadow uses `shadow-card` token — needs to be heavier
- No directional right/bottom border dimming
- backdrop-blur is `3xl` (64px) — needs upgrade to `[40px]` explicit for consistency

**Target state:**
- All `<Card>` components: `bg-white/[0.02] backdrop-blur-[40px]`
- 3D light-catch bevel: `border-t border-l border-white/10 border-r-white/5 border-b-white/5`
  (top/left brighter = light source from top-left; right/bottom dimmer = shadow side)
- Heavy resting shadow: `shadow-[0_24px_48px_-12px_rgba(0,0,0,0.8)]`
- Consistent depth hierarchy: cards float, content sits on cards, overlays above both
</objective>
