## Data Flow

1. **User Request:** Frontend sends `RouteRequest` (source, dest, EV state) to `/api/generate-route`.
2. **Pathfinding:** `backend_api.py` delegates to `RouteGenerator`.
3. **Optimization:** `RouteGenerator` considers:
    - Road network topology (`RoadGraph`)
    - Battery constraints (`EVState`)
    - Traffic conditions (from GANs)
4. **Response:** Returns optimal route candidates with metrics (distance, energy, time).
